Searchbar

Мы объединили два блока - input и button, поработали над стилями и получили блок для поиска. Блоки input, button не знают о том как располагаться друг относительно друга, поэтому за это отвечает блок searchbar у которого есть элементы searchbar__input, searchbar__button 

          
            <div class="searchbar">
              <input class="input  searchbar__input" placeholder="Поиск" type="text" value="" />
              <button class="button button_color_primary searchbar__button">Поиск</button>
            </div>
          
          
          
            .searchbar {
              display: flex;
              width: 250px;
            }

            .searchbar__button {
              border-top-left-radius: 0;
              border-bottom-left-radius: 0;
            }

            .searchbar__button:hover {
              z-index: 1;
            }

            .searchbar__input {
              border-top-right-radius: 0;
              border-bottom-right-radius: 0;
              border-right: 0;
            }

Form

Объединив пару инпутов и кнопок получаем блок формы. Обратите внимание как мы указываем расположение блока относительно других блоков. Так как напрямую блоку задавать отступы нельзя по БЭМ, отступы задает родитель, использующий этот блок. В данном примере у нас есть блок sign-in-form и у него есть два инпута и кнопка, которые являются элементами данного блока. И именно элементам мы задаем внешнюю геометрию. Таким образом блоки input, button остаются независимыми а за их расположение друг относительно друга отвечает блок, который использует их (sign-in-form).

Добро пожаловать! 
          
            <form class="sign-in-form">
              <legend class="sign-in-form__legend">Добро пожаловать!</legend>
              <div class="sign-in-form__body">
                <input class="input sign-in-form__input" placeholder="Логин" type="text" value="" />
                <input class="input sign-in-form__input" placeholder="Пароль" type="password" value="" />
              </div>
              <button class="button button_color_primary sign-in-form__button">Войти</button>
            </form>
          
          
          
            .sign-in-form {
              border: 1px solid #eee;
              width: 250px;
              padding: 10px;
              border-radius: 8px;
            }

            .sign-in-form__legend {
              text-align: center;
              margin-bottom: 16px;
              font-weight: bold;
            }

            .sign-in-form__input {
              width: 100%;
              margin-bottom: 8px;
            }

            .sign-in-form__input:last-child {
              margin: 0;
            }

            .sign-in-form__body {
              margin-bottom: 32px;
            }

            .sign-in-form__button {
              width: 100%;
            }
